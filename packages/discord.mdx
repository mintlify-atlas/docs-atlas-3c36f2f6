## Overview

`@magicyan/discord` provides a comprehensive set of helper functions to simplify Discord bot development with discord.js. It includes utilities for channels, members, roles, embeds, components, messages, and more.

## Installation

<CodeGroup>
```bash npm
npm install discord.js @magicyan/discord
```

```bash yarn
yarn add discord.js @magicyan/discord
```

```bash pnpm
pnpm add discord.js @magicyan/discord
```
</CodeGroup>

<Note>
  Requires **discord.js ^14.25.1** as a peer dependency. Also includes all utilities from `@magicyan/core`.
</Note>

## Key Features

### Channel Utilities

Find channels with flexible filtering options.

```typescript
import { findChannel } from '@magicyan/discord';

// Find by ID
const channel = findChannel(guild).byId('123456789');

// Find by name
const general = findChannel(guild).byName('general');

// Find in category
const ticket = findChannel(guild).inCategoryName('Support').byName('ticket-1');

// Custom filter
const voiceChannels = findChannel(guild).byFilter(ch => ch.isVoiceBased());
```

[View Channel API →](/api/discord/channels)

### Member Utilities

Find guild members with various criteria.

```typescript
import { findMember } from '@magicyan/discord';

// Find by username
const member = findMember(guild).byUsername('JohnDoe');

// Find by display name
const member2 = findMember(guild).byDisplayName('John');

// Find by ID
const member3 = findMember(guild).byId('123456789');

// Custom filter
const admins = findMember(guild).byFilter(m => m.permissions.has('Administrator'));
```

[View Member API →](/api/discord/members)

### Role Utilities

Find and manage roles easily.

```typescript
import { findRole } from '@magicyan/discord';
```

[View Role API →](/api/discord/roles)

### Embed Builder

Enhanced embed creation with extended functionality.

```typescript
import { createEmbed } from '@magicyan/discord';

// Create a rich embed
const embed = createEmbed({
  title: 'Welcome!',
  description: [
    'Welcome to our server!',
    'Please read the rules.',
  ],
  color: 'Blue',
  timestamp: new Date(),
  footer: { text: 'Server Bot', iconURL: guild.iconURL() },
  thumbnail: member.displayAvatarURL(),
  fields: [
    { name: 'Members', value: guild.memberCount.toString(), inline: true },
    { name: 'Channels', value: guild.channels.cache.size.toString(), inline: true },
  ],
});

await channel.send({ embeds: [embed] });
```

[View Embed API →](/api/discord/embeds)

### Component Utilities

Create buttons, modals, and action rows easily.

```typescript
import { 
  createRow, 
  createLinkButton, 
  createModal, 
  createTextInput 
} from '@magicyan/discord';

// Create buttons
const row = createRow(
  new ButtonBuilder({
    customId: 'accept',
    label: 'Accept',
    style: ButtonStyle.Success,
  }),
  createLinkButton({
    label: 'Documentation',
    url: 'https://docs.example.com',
  })
);

// Create modals
const modal = createModal({
  customId: 'feedback',
  title: 'Feedback Form',
  components: [
    createRow(
      createTextInput({
        customId: 'message',
        label: 'Your Feedback',
        style: TextInputStyle.Paragraph,
        required: true,
      })
    ),
  ],
});

await interaction.showModal(modal);
```

[View Component API →](/api/discord/components)

### Message Utilities

Find and parse messages with ease.

```typescript
import { findMessage, getMessageURLInfo, fetchMessageFromURL } from '@magicyan/discord';

// Find messages
const messages = await findMessage(channel).all({ limit: 10 });
const message = await findMessage(channel).byId('123456789');

// Parse message URLs
const info = getMessageURLInfo('https://discord.com/channels/guild/channel/message');
console.log(info); // { guildId, channelId, messageId }

// Fetch from URL
const message = await fetchMessageFromURL(client, 'https://discord.com/...');
```

## Type Guards

Type-safe guards for Discord objects:

```typescript
import { isAttachment, isButton, isModal, isMessage } from '@magicyan/discord';

if (isButton(component)) {
  // TypeScript knows this is a ButtonComponent
  await component.reply('Button clicked!');
}

if (isMessage(interaction)) {
  // TypeScript knows this is a Message
  await interaction.reply('Message received!');
}
```

## Usage Example

Here's a complete example using multiple utilities:

```typescript
import { 
  Client, 
  GatewayIntentBits,
  Events,
  ButtonStyle,
  ButtonBuilder,
} from 'discord.js';
import { 
  CustomItents,
  findChannel,
  createEmbed,
  createRow,
  brBuilder,
} from '@magicyan/discord';

const client = new Client({ intents: CustomItents.All });

client.on(Events.ClientReady, async (readyClient) => {
  console.log(`Logged in as ${readyClient.user.tag}`);
});

client.on(Events.GuildMemberAdd, async (member) => {
  const channel = findChannel(member.guild).byName('welcome');
  if (!channel?.isTextBased()) return;

  const embed = createEmbed({
    title: 'Welcome!',
    description: brBuilder(
      `Welcome ${member.user}, to **${member.guild.name}**!`,
      '',
      'Please read the rules and have fun!'
    ),
    color: 'Green',
    thumbnail: member.displayAvatarURL(),
    timestamp: new Date(),
  });

  const row = createRow(
    new ButtonBuilder({
      customId: 'read-rules',
      label: 'Read Rules',
      style: ButtonStyle.Primary,
    })
  );

  await channel.send({ embeds: [embed], components: [row] });
});

client.login(process.env.DISCORD_TOKEN);
```
