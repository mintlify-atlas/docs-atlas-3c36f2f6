## Welcome to Magicyan

Magicyan is a comprehensive TypeScript SDK that simplifies Discord bot development by providing utility functions, helper classes, and extended events for discord.js. Built as a monorepo, Magicyan offers specialized packages that work together seamlessly to enhance your Discord bot development experience.

## Why Magicyan?

Building Discord bots with discord.js can be repetitive and verbose. Magicyan eliminates boilerplate code and provides intuitive helpers for common tasks:

- **Type-safe utilities** - Full TypeScript support with intelligent type inference
- **Simplified API** - Clean, intuitive functions that wrap complex discord.js operations
- **Extended events** - Additional events beyond discord.js defaults (timeouts, kicks, role changes)
- **Component builders** - Easy-to-use builders for embeds, modals, and UI components
- **Flexible architecture** - Use only the packages you need

## Key Features

### Simplified Channel Management

Find channels easily with intuitive methods:

```typescript
const channel = findChannel(guild, ChannelType.GuildText)
  .byName("general");
```

### Enhanced Embed Builder

Create rich embeds with less code:

```typescript
const embed = createEmbed({
  title: "Welcome!",
  description: ["Line 1", "Line 2"],
  color: "#5865F2",
  timestamp: true,
  footer: "Powered by Magicyan"
});
```

### Extended Events

Listen to events that discord.js doesn't provide:

```typescript
client.on("guildMemberTimeoutAdd", (member, timeout) => {
  console.log(`${member.user.tag} was timed out`);
});
```

### Custom Intents Presets

Quickly configure your bot with predefined intent groups:

```typescript
const client = new Client({
  intents: CustomItents.All
});
```
