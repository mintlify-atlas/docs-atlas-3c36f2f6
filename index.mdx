## Welcome to Magicyan

Magicyan is a collection of TypeScript utility libraries designed to simplify and enhance Discord bot development with discord.js. Build powerful Discord bots with less boilerplate and more functionality.

## Featured Packages

<Accordion title="@magicyan/core">
  Lightweight utility functions for common tasks including formatting, validation, timers, date manipulation, and mathematical operations.
</Accordion>

<Accordion title="@magicyan/discord">
  Comprehensive helper functions for Discord.js including channel management, member operations, role utilities, embed builders, and component creation.
</Accordion>

<Accordion title="@magicyan/discord-events">
  Extended event system for Discord.js providing additional events such as guild member voice state changes, timeout management, and enhanced audit log events.
</Accordion>

<Accordion title="@magicyan/discord-ui">
  Interactive UI components including navigation menus and user prompts to create rich Discord bot experiences.
</Accordion>
