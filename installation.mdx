## Prerequisites

Before installing Magicyan, ensure you have:

- **Node.js** 18 or higher
- **discord.js** 14.25.1 or higher (peer dependency)
- A package manager: npm, yarn, or pnpm

<Note>
Magicyan requires discord.js as a peer dependency. Make sure to install discord.js in your project.
</Note>

## Install discord.js

If you haven't already, install discord.js first:

<CodeGroup>
```bash npm
npm install discord.js@^14.25.1
```

```bash yarn
yarn add discord.js@^14.25.1
```

```bash pnpm
pnpm add discord.js@^14.25.1
```
</CodeGroup>

## Install Magicyan Packages

### Core Package (Required)

Install the main Discord helper package:

<CodeGroup>
```bash npm
npm install @magicyan/discord
```

```bash yarn
yarn add @magicyan/discord
```

```bash pnpm
pnpm add @magicyan/discord
```
</CodeGroup>

<Info>
The `@magicyan/discord` package automatically includes `@magicyan/core` as a dependency.
</Info>

### Extended Events (Optional)

For additional Discord events beyond discord.js defaults:

<CodeGroup>
```bash npm
npm install @magicyan/discord-events
```

```bash yarn
yarn add @magicyan/discord-events
```

```bash pnpm
pnpm add @magicyan/discord-events
```
</CodeGroup>

### UI Components (Optional)

For advanced Discord UI components:

<CodeGroup>
```bash npm
npm install @magicyan/discord-ui
```

```bash yarn
yarn add @magicyan/discord-ui
```

```bash pnpm
pnpm add @magicyan/discord-ui
```
</CodeGroup>

### Canvas Utilities (Optional)

For image generation with canvas:

<CodeGroup>
```bash npm
npm install @magicyan/canvas @napi-rs/canvas
```

```bash yarn
yarn add @magicyan/canvas @napi-rs/canvas
```

```bash pnpm
pnpm add @magicyan/canvas @napi-rs/canvas
```
</CodeGroup>

<Note>
`@magicyan/canvas` requires `@napi-rs/canvas` as a peer dependency.
</Note>

### Minecraft Utilities (Optional)

For fetching Minecraft API data:

<CodeGroup>
```bash npm
npm install @magicyan/minecraft
```

```bash yarn
yarn add @magicyan/minecraft
```

```bash pnpm
pnpm add @magicyan/minecraft
```
</CodeGroup>

## Package Overview

Choose the packages that fit your needs:

| Package | Use Case | Required |
|---------|----------|----------|
| `@magicyan/discord` | Core Discord helpers (channels, embeds, components) | Yes |
| `@magicyan/core` | Utility functions (included with discord package) | Auto-installed |
| `@magicyan/discord-events` | Extended events (timeouts, kicks, voice) | Optional |
| `@magicyan/discord-ui` | Advanced UI components | Optional |
| `@magicyan/canvas` | Image generation | Optional |
| `@magicyan/minecraft` | Minecraft API integration | Optional |

### Recommended Installation

For most Discord bots, we recommend:

<CodeGroup>
```bash npm
npm install discord.js @magicyan/discord @magicyan/discord-events
```

```bash yarn
yarn add discord.js @magicyan/discord @magicyan/discord-events
```

```bash pnpm
pnpm add discord.js @magicyan/discord @magicyan/discord-events
```
</CodeGroup>

## TypeScript Configuration

Magicyan is built with TypeScript and requires proper configuration. Create or update your `tsconfig.json`:

```json tsconfig.json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "node",
    "esModuleInterop": true,
    "strict": true,
    "skipLibCheck": true,
    "resolveJsonModule": true,
    "outDir": "./dist",
    "rootDir": "./src"
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules"]
}
```

<Tip>
Magicyan uses ES modules. Set `"type": "module"` in your `package.json` for the best experience.
</Tip>

## Package.json Setup

Update your `package.json` to use ES modules:

```json package.json
{
  "name": "my-discord-bot",
  "version": "1.0.0",
  "type": "module",
  "scripts": {
    "start": "node dist/index.js",
    "build": "tsc",
    "dev": "tsx src/index.ts"
  },
  "dependencies": {
    "discord.js": "^14.25.1",
    "@magicyan/discord": "^1.7.2",
    "@magicyan/discord-events": "^1.2.1"
  },
  "devDependencies": {
    "typescript": "^5.5.4",
    "tsx": "^4.19.3"
  }
}
```

## Verify Installation

Create a simple test file to verify your installation:

```typescript test.ts
import { Client } from "discord.js";
import { CustomItents, createEmbed } from "@magicyan/discord";

console.log("Magicyan installed successfully!");

const client = new Client({
  intents: CustomItents.Guild
});

const embed = createEmbed({
  title: "Installation Complete",
  description: "Magicyan is ready to use!",
  color: "#00FF00"
});

console.log("Embed created:", embed.data.title);
```

Run it with:

```bash
tsx test.ts
```

If you see "Magicyan installed successfully!" and "Embed created: Installation Complete", you're all set!
